# 机器学习可视化平台

## 1. 项目介绍

* 前端使用 Vue
* 后端使用 Django

* 目前已实现：
    * 用户注册、登录、登出
    * 前端框架
    * 后端框架
    * 线性回归可视化

* TODO：
    * 其他算法的可视化
    * 管理员功能（非必要）

* 运行
    * 前端
        * `cd frontend`
        * `npm install` or `cnpm install`
        * `npm run dev`
    * 后端
        * `cd ML_visual_backend`
        * `pip install -r requirements.txt`
        * `python manage.py migrate`
        * `python manage.py runserver localhost:8080`
